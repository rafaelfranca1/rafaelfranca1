# 👋 Olá!
Sou Rafael de França, estudante de Ciência da Computação com foco em engenharia de software, movido pela curiosidade de entender como as coisas funcionam nos bastidores da tecnologia. Minha atuação é pautada pelo desenvolvimento de sistemas robustos e arquiteturas bem estruturadas, onde priorizo o código limpo e a facilidade de manutenção para garantir soluções duradouras e escaláveis. Com experiência prática em sistemas empresariais e APIs, busco resolver problemas complexos de forma analítica, criando ferramentas que gerem impacto positivo real para pessoas e negócios. Sou um entusiasta da evolução contínua e busco a excelência técnica em cada projeto, mantendo uma visão criativa para encontrar caminhos inovadores na área.
